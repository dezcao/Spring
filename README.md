# Spring legacy MVC project

### 개발버전
    - Java8, Apache Tomcat8.5, Spring4, Maven4.0, MyBatis3.4, H2DB1.4, Apache Tiles3, Twitter Bootstrap v3.3.7

### 구현기능
    - XSS Filter, google login(security), board

### 실행필수
    - lombok install, h2db start
